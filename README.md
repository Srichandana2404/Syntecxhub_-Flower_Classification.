# Syntecxhub_Flower_Classification.
1. 	Project Title
Flower Classification using Logistic Regression and Decision Tree

2. 	Features
• 	Data preprocessing (handling duplicates, nulls, etc.)
• 	Exploratory data analysis (scatter plots, pair plots)
• 	Model building (Logistic Regression, Decision Tree)
• 	Model evaluation (accuracy score, confusion matrix, classification report)
• 	Predictions on new samples

4. 	Results
• 	Both models achieved 96% and 100% accuracy respectively on the test set.

5. 	Tech Stack
• 	Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
